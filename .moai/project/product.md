# StudioX v1 - 제품 문서

## 프로젝트 개요

**프로젝트명**: StudioX v1 - AI 노드 기반 창의 스튜디오

**현재 버전**: 1.0.0

**개발 단계**: 활발한 개발 중

**기술 스택**: TypeScript 5.8.2 + React 19.2.0 + Vite 6.2.0

---

## 핵심 설명

StudioX v1은 Google Gemini AI와 실시간 노드 기반 워크플로우를 결합한 **무한 캔버스 비주얼 노드 에디터**입니다. 사용자는 30개 이상의 다양한 노드를 조합하여 AI 기반 이미지/비디오 생성 파이프라인을 직관적으로 설계하고 실행할 수 있습니다.

### 솔루션의 핵심 가치

**창의적 자동화**: 복잡한 멀티스텝 이미지/비디오 생성 워크플로우를 노드 드래그앤드롭으로 구성하고 즉시 실행

**AI 통합**: Google Gemini API를 통한 고급 이미지 생성, 배경 제거, 이미지 편집, 비디오 생성 등 다양한 AI 기능

**무한 확장성**: D3 기반 무한 캔버스로 제한 없이 노드와 워크플로우 추가 가능

**패션/상품 전문**: Virtual Try-On(VTON), GridShot 등 패션 및 상품 촬영에 최적화된 워크플로우 지원

---

## 핵심 기능

### 비주얼 노드 에디터

- **30개 이상의 노드 타입**: 이미지, 비디오, 텍스트, AI 어시스턴트, 이미지 편집/로드/수정/미리보기, 비디오 로드/스티칭, 모델 관리, Virtual Try-On, 카메라, 프리셋, 주석, 스토리보드, 스크립트, 배열/리스트, 프롬프트 연결, 합성, 배경제거, 그룹, 그리드샷, 아웃핏 디테일 등
- **무한 캔버스**: D3 기반 팬/줌 기능으로 제약 없이 워크플로우 확장
- **직관적 인터페이스**: 노드 드래그앤드롭, 연결선 드로우, 실시간 입력값 변경

### AI 기반 생성 기능

- **이미지 생성**: Google Gemini의 고급 프롬프트 기반 이미지 생성
- **이미지 편집**: 기존 이미지에 대한 배경 제거, 수정, 합성 작업
- **비디오 생성**: 텍스트-비디오 또는 이미지-비디오 변환
- **Virtual Try-On**: 패션 제품의 가상 착용 시뮬레이션
- **배경 제거 (RMBG)**: 고품질 배경 제거 AI 기능

### 워크플로우 관리

- **로컬 프로젝트 저장**: IndexedDB 기반 로컬 데이터베이스로 프로젝트 자동 저장
- **히스토리 관리**: 실행 히스토리 추적 및 이전 단계로의 복원
- **노드 제어**: 노드 Bypass/Collapse 기능으로 워크플로우 디버깅 및 최적화
- **패턴 프리셋**: 자주 사용하는 워크플로우 패턴을 프리셋으로 저장 및 재사용

### 고급 기능

- **스토리보드/스크립트**: 영상 제작을 위한 스토리보드 및 스크립트 관리
- **배열/리스트 처리**: 대량의 이미지/비디오 배치 처리
- **커스텀 모델 지원**: 사용자 정의 AI 모델 연동
- **카메라 시뮬레이션**: 카메라 각도, 조명 등 세부 설정 조정

## 타겟 사용자

### 1차 대상
- **패션/상품 디자이너**: 상품 촬영 및 시각화 자동화 필요

## 주요 사용 사례

### 사용 사례 1: 패션 제품 촬영 자동화

**상황**: 온라인 쇼핑몰에서 옷 상품의 다양한 각도 사진이 필요

**워크플로우**:
1. 상품 원본 이미지 로드
2. 배경 제거 (RMBG)
3. Virtual Try-On 노드로 다양한 모델에 착용
4. 조명/필터 조정
5. 최종 이미지 배치 내보내기

**효과**: 수동 촬영 대비 90% 시간 단축, 일관된 스타일 유지

### 사용 사례 2: AI 기반 마케팅 콘텐츠 생성

**상황**: 광고 캠페인을 위한 다양한 비주얼 콘텐츠 필요

**워크플로우**:
1. 마케팅 텍스트 입력
2. AI 이미지 생성 노드로 컨셉 이미지 생성
3. 이미지 편집/수정으로 세부 조정
4. 비디오 생성 노드로 애니메이션 광고 제작
5. 스토리보드로 멀티플 시나리오 구성

**효과**: 외부 제작사 의존도 감소, 빠른 반복 가능

### 사용 사례 3: 스토리보드 기반 영상 제작

**상황**: 장편 광고 영상 또는 영화 촬영 계획이 필요

**워크플로우**:
1. 스토리보드 노드로 장면별 시나리오 정의
2. 각 장면별 배경/캐릭터 이미지 AI 생성
3. 배열 처리로 연속 장면 생성
4. 비디오 스티칭으로 최종 영상 편집

**효과**: 사전 시각화를 통한 촬영 효율화, 수정 비용 절감

### 사용 사례 4: 대량 배치 처리

**상황**: 수백 개의 상품에 대한 자동화된 이미지 생성

**워크플로우**:
1. 상품 목록을 배열/리스트 노드로 정의
2. 각 상품별로 반복 처리 설정
3. 자동 이미지 생성 및 배경 제거 파이프라인
4. 모든 결과 일괄 내보내기

**효과**: 수천 개 이미지를 자동으로 생성, 수동 작업 완전 제거
