# AI-StudioX 제품 정보

## 프로젝트 개요

**프로젝트명**: AI-StudioX
**버전**: 1.0.0
**배포 URL**: https://ai-studio-x-lime.vercel.app
**업데이트**: 2026년 3월 13일

## 설명

무한 캔버스 위에서 에이전트와 노드를 사용하여 이미지, 비디오, 텍스트 콘텐츠를 생성하는 비주얼 AI 워크플로우 앱입니다. 사용자는 직관적인 인터페이스로 복잡한 AI 작업을 시각적으로 구성하고 실행할 수 있습니다.

## 대상 사용자

- AI 콘텐츠 창작자
- 디자이너
- 마케터

## 핵심 기능

### 1. 무한 캔버스
D3.js 기반의 전체 팬(Pan)과 줌(Zoom) 기능을 제공하는 무한 캔버스 환경입니다. 사용자는 제약 없이 캔버스를 탐색하고 노드를 배치할 수 있습니다.

### 2. 27개 노드 타입
다양한 작업을 지원하는 풍부한 노드 타입 시스템:

**기본 노드**
- Text: 텍스트 입력 및 처리
- Assistant: AI 어시스턴트 통합
- Image: 이미지 처리
- Video: 비디오 처리

**AI 생성 노드**
- Camera: 카메라 입력
- Model: AI 모델 실행
- Vton: 가상 피팅 (Virtual Try-On)
- ImageModify: 이미지 수정 및 변환

**콘텐츠 구성 노드**
- Storyboard: 스토리보드 생성
- Script: 스크립트 작성
- Comment: 주석 및 메모
- Composite: 여러 콘텐츠 조합

**데이터 처리 노드**
- Array: 배열 처리
- List: 리스트 관리
- Stitch: 이미지/영상 연결
- GridShot: 그리드 기반 이미지 추출
- GridExtractor: 그리드 데이터 추출
- Group: 노드 그룹화

**추가 노드**
- 사용자 요구에 따라 확장 가능

### 3. 노드 간 타입 기반 연결 시스템
강력한 타입 검증을 통해 호환되는 노드만 연결할 수 있으며, 데이터 흐름의 정합성을 보장합니다.

### 4. Google Gemini API 통합
최신 Google Gemini API를 사용하여:
- 이미지 생성
- 비디오 생성
- 텍스트 생성

### 5. IndexedDB 기반 프로젝트 로컬 영속화
사용자의 프로젝트를 브라우저의 IndexedDB에 저장하여 로컬에서 안전하게 관리할 수 있습니다.

### 6. History Panel
생성된 모든 에셋의 이력을 관리하고 추적할 수 있는 전용 패널입니다.

### 7. Playground Modal
AI 기능을 테스트하고 실험할 수 있는 전용 인터페이스입니다.

## 사용 사례

### 1. AI 패션 모델 이미지 생성
Model 노드를 사용하여 다양한 스타일의 패션 모델 이미지를 생성할 수 있습니다.

### 2. 가상 피팅 (Virtual Try-On)
Vton 노드를 통해 사용자가 의류 상품을 가상으로 시착해볼 수 있는 이미지를 생성합니다.

### 3. 광고 스토리보드 생성
Storyboard 노드를 활용하여 광고 캠페인의 스토리보드를 자동 생성합니다.

### 4. 상품 스크립트 + 씬 이미지 파이프라인
Script 노드에서 상품 소개 스크립트를 작성하고, 이를 바탕으로 해당하는 씬 이미지를 자동 생성하는 통합 워크플로우입니다.
